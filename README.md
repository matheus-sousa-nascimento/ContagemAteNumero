# Contagem Até Número

Este é um programa simples em Java que imprime os números de 1 até o número positivo fornecido pelo usuário. O programa garante que o número fornecido seja positivo antes de proceder com a execução.

## Funcionalidade

1. O usuário insere um número positivo.
2. O programa imprime os números de 1 até o número informado pelo usuário.
3. Caso o número inserido seja zero ou negativo, o programa pede para o usuário inserir um número positivo.
